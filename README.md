# 🐾NekoProject (ฅ>ω<*ฅ)
一个简单的游戏服务端的 Docker Desktop 管理模型。。

## 1. 硬件要求

- CPU 架构为 AMD64 并支持“二级地址转换技术”（SLAT）。
- CPU 支持虚拟化技术（VT-x 或 AMD-V）。
- 32G 以上的物理内存。

## 2. 系统要求

- Windows 11 专业版（64 位）以上版本。
- 请将 Windows 更新到最新稳定版本。
